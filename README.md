# Fiona的ero教室

​	可爱的fiona，今天也在教室里过着日复一日的日常。突然有一天，她从老师的讲桌下面拿到个不可思议的宝箱(H的道具箱)，从此她开始了不同寻常的每一天....





###### 故事设定

> Player
>
> Fiona原本是个可可爱爱的同学，乖巧又活泼。一天，她无意间打开了神奇的**百宝箱**，发现里面装着各种奇妙的道具。第一个道具让她的情绪变得格外**敏感**，好奇心和调皮心像火苗一样被点燃，她开始对Player表现出比平时更多的依赖和撒娇。
>
> 第二天，Player自己也拿到了一个道具——**隐身斗篷**。穿上它，他可以悄无声息地观察Fiona。课堂上、教室角落，Fiona在不知情的情况下玩起了道具小游戏，Player则在暗中观察，偶尔偷偷进行一些恶作剧——比如把她手里的魔笔换成闪亮的笔芯，或者在她玩小道具时轻轻改变道具的效果。每一次道具互动都让Fiona微微惊讶、脸红心跳，带着娇羞又不失俏皮的反应，让Player的好奇心不断被挑起。
>
> 第三天，Fiona又得到一个新道具——一个会动的**可变式魔法小道具**。她尝试着探索道具的用法，时不时发出轻快的笑声和小小的抱怨，而Player仍旧在暗中观察，有时会悄悄改变道具，让Fiona的反应更加意外和可爱。每一天，宝箱里的奇妙道具都像在推动着两人的关系，暧昧的气氛在校园里悄悄升温，而Fiona也慢慢展现出她调皮、挑逗又魅惑的一面——不失童趣，却让人心动不已。
>
> 第四天，Player也拿到**催眠魔法**，可以和Fiona进行互动游戏，测试她的机智与反应，也让两人的关系更加亲密。每一次互动都是小小的试探和心跳加速，让校园生活变得充满了不可预测的乐趣。
